# Inactive Employees

## Summary
Show today's active employee absences in Intranet Home.
Similar view to Events webpart

## View Requirements

Create a list with the following columns:

| Internal Name   | Type               |
|-----------------|--------------------|
| **Colaborador**       | People |
| **Inicio**          | Date |
| **Fin**          | Date |
| **Motive**          | Single Text |
